Metodologia:

Para este estudo, foram analisados um total de 21 campeonatos de PUBG, sendo selecionados 3 campeonatos de cada servidor...

Objetivo:

O objetivo deste relatório é analisar e comparar as principais diferenças de estilo de jogo dos principais servidores do mundo em campeonatos de PUBG...

Conceitos Importantes:

  • PPG = Points per game / Pontos por Partida

  • CPG = Position points per game / Pontos de Colocação por partida
